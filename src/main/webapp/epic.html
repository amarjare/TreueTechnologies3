Do Epic Shit

Do Epic Shit is a self-help book that is written for new readers because Ankur Warikoo has only shared basic information which is already out there on the internet for free but as this is written in simple ENGLISH, this might be a good start for new readers. I’ve been following Ankur Warikoo for about a year now and when I come to know, that he has announced the book, I immediately pre-ordered it and it took me only a day to finish. I usually don’t complete any book in a day but this was a bit different because it was about failures. I believe It’s better to learn from someone else’s mistake rather than repeating the same in our life. Ankur Warikoo has shared his mistakes about money, business, and time.

AUTHOR : Ankur Warikoo

PRICE : 280 Rs.

Rating: 4.5/5
